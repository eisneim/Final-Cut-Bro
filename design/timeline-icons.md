# 时间线工具栏图标规范

> 来源:对 `design/时间线工具截图.jpg` 与 `design/toolbar dropdonw.jpg` 放大后视觉识别。
> 实现为 SwiftUI `Shape`/`Path`,描边 ~1.5pt,色用 `Tokens.Palette.textIcon`(#EEEEEE),选中态 `Tokens.Palette.selectYellow`。
> 画布 24×24,glyph 居中约 16×16。FCPX 扁平线性风格,无渐变阴影。

## 四个编辑按钮(工具栏左侧一组)
| 名称 | 动作 | glyph |
|---|---|---|
| 连接 Connect | 把所选源连接到主情节(connected) | 两个**描边**矩形上下堆叠,上方矩形约为下方 60% 宽、居中,二者间留小缝(下=主轴,上=连接片段) |
| 插入 Insert | 插入到主情节(在播放头处) | 一个**描边**矩形 + 一根向下箭头从顶部插入,箭头头部止于矩形垂直中心 |
| 追加 Append | 追加到主情节末尾 | 一个**描边**矩形 + 向下箭头,箭头头部止于矩形**底边**(落在末尾之意) |
| 覆盖 Overwrite | 覆盖(在播放头处) | 一个**实心填充**矩形 + 向下箭头插入顶部(用填充区别于插入/追加) |

之后接一个向下 chevron(编辑模式菜单),再接工具选择器(下方)。

## 工具选择下拉(默认=选择)
| 工具 | 快捷键 | glyph |
|---|---|---|
| 选择 Select | A | 经典箭头光标:空心三角指针,尖朝左上,右下有短小尾柄 |
| 修剪 Trim | T | 左右两个相向三角箭头,中间一对竖线(`◀‖‖▶`) |
| 位置 Position | P | 实心向右三角旗(左竖边 + 右尖),无尾 |
| 范围选择 Range | R | 左侧圆角方框(右边为虚线)+ 右侧实心向右箭头叠在方框右缘 |
| 切割 Blade | B | 剃刀刀片(FCPX 用 razor):一个平行四边形刀片 + 斜切刃口(非剪刀) |
| 缩放 Zoom | Z | 放大镜:圆 + 4~5 点钟方向 45° 直柄 |
| 手 Hand | H | 张开手掌:四指 + 左上拇指 + 圆角掌根 |

> 工具栏布局(替换现有"索引 ✛⊟✄ ... 主时间轴 ... "):
> `[连接][插入][追加][覆盖] ⌄   [工具▾]      <flex>      [缩放−][缩放条/+]   时间码`
> 不要"索引"二字;尺寸正常(按钮约 24~28pt 高,图标 16pt)。
